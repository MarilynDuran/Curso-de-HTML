¿Que es open Graph?

Es un protocolo de internet que fue creado por facebook


Para estandarizar el uso de los metadatos en una pagina web

 
Permite  a los desarrollar3es proporcionar metaetiquetas en el codigo HTML
EJEMPLO: Como enlaces, titulos, imagenes, datos relevantes.


    Open graph hace que cuando un enlace se comparta en redes la plataforma pueda mostrar una vista previa. 
    - Facebook

Etiquetas

  • og: title : Titulo de la pagina
  • og: type : El tipo del sitio web
  • og: image : Enlace de una imagen
  • og: url : Url de la pagina actual
